🌳 Grow Wealth Wild (Money Forest)
A gamified personal finance web application that transforms your financial habits into a living, evolving forest.

🌐 Live Demo: https://grow-wealth-wild.base44.app

💡 Overview
Grow Wealth Wild reimagines personal finance for Gen Z and young adults by turning money management into a visual and emotional experience.

Instead of traditional dashboards filled with numbers and charts, the app represents your financial behavior as a dynamic forest ecosystem:

🌱 Saving money grows trees
🌸 Achieving goals makes flowers bloom
🦋 Consistent habits bring life (birds, butterflies)
🌿 Overspending creates weeds
🌦 Financial health affects the entire environment
The goal is to make managing money feel rewarding, intuitive, and engaging rather than stressful.

✨ Key Features
🔐 User Authentication (Sign Up / Login flow)
💸 Income & Expense Tracking
🎯 Goal-Based Financial Trees
🌳 Interactive Forest Dashboard
📊 Financial Health Visualization
🌿 Visual Feedback System (Good vs Bad Habits)
📈 Monthly Insights & Reports
⚡ Fast and responsive UI
🧠 Problem Statement
Most personal finance applications are:

Complex and overwhelming
Data-heavy and boring
Emotionally disconnected from users
This leads to poor financial habits and lack of engagement.

✅ Solution
Grow Wealth Wild solves this by combining:

Gamification
Behavioral psychology
Visual storytelling
to create a system where:

“Users grow a forest instead of managing a spreadsheet.”

🛠️ Tech Stack
Frontend: React + Vite
Styling: Tailwind CSS
Backend: Base44
Deployment: Base44 Hosting
🧩 System Architecture
User Input (Income / Expense / Goals) ↓ Backend Processing (Base44) ↓ Financial Logic Engine ↓ Forest Visualization System ↓ Interactive Dashboard
